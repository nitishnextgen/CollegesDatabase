## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- No additional dependencies required - runs entirely in the browser

### Installation

1. Clone or download the project
2. Navigate to the `CollegesDatabase` directory
3. Open `index.html` in your web browser

```bash
# Simply open the file in your browser
open index.html  # macOS
start index.html # Windows
xdg-open index.html # Linux
```

## Usage

### Search Universities
1. Navigate to the Search section
2. Enter keywords in the search bar
3. Apply filters by state, region, or stream
4. Browse the filtered results

### Compare Universities
1. Go to the Compare section
2. Select universities from the dropdown menus
3. View side-by-side comparison of key details

### View Statistics
1. Visit the Statistics section
2. Explore interactive charts and insights
3. Understand distribution trends across regions and streams

### Download Data
1. Click the Download button in the navigation bar
2. Export the complete database to Excel format

## Technologies Used
